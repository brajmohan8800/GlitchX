🎬 The Process

How I approach testing

The Software Testing Life Cycle I follow, step by step, on every project.

1
Requirement Analysis
Understand what a feature should do — and what it shouldn't.
2
Test Planning
Decide scope, tools and priorities before writing a case.
3
Test Case Design
Clear, repeatable cases — happy paths and edge cases both.
4
Test Execution
Run cases manually or via automation, and log everything.
5
Defect Reporting
Clean, reproducible bug reports developers want to read.
6
Regression & Closure
Re-test fixes, confirm nothing broke, then sign off.
